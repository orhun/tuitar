# Assembling Tuitar v0

## Bill of Materials

- ESP-WROOM-32 Devkit
- TFT 1.8 ST7735 128X160 Display
- MP1584 Buck Converter
- MAX4466
- LM358P
- On/off switch
- 2 x buttons
- 2 x 1M pot
- 3 x 1K resistor
- 2 x 10K resistor
- LED (red)
- 1N5819 diode
- 1uF capacitor
- Mono jack
- A kickass electric guitar

## Case

The case dimensions are 100mm x 65mm x 50mm (L x W x H).

It needs 9 holes:

- Display
- 2 x pots
- 2 x buttons
- On/off switch
- Microphone input
- Jack input
- USB connection
